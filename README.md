## Project Overview
KYC Shield is an AI-powered digital identity verification platform designed to perform secure KYC (Know Your Customer) verification.

The platform allows users to upload identity documents and perform liveness checks. AI models analyze the submitted data to detect potential fraud signals such as document tampering, deepfakes, or mismatched identities.

This system helps automate identity verification processes for fintech platforms, digital banks, and online services requiring secure user authentication.

## Setup Instructions
Follow the steps below to run the project locally.

### 1. Clone the repository
```bash
git clone https://github.com/your-username/kyc-shield.git
```

### 2. Navigate to the project directory
```bash
cd kyc-shield
```

### 3. Install dependencies
```bash
npm install
```

### 4. Configure environment variables
Create a `.env` file and add:
```env
GEMINI_API_KEY=your_api_key
JWT_SECRET=your_secret
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
```

### 5. Run the development server
```bash
npm run dev
```

Open the browser:
http://localhost:5173
